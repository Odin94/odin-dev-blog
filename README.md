# Odin Matthias — Developer Blog

Personal portfolio and developer blog built with Astro, React, and Tailwind CSS.

## Run locally

Requires Node.js and pnpm.

```sh
pnpm install
pnpm dev
```

The development server runs at `http://localhost:4321` by default.

## Build

```sh
pnpm build
pnpm preview
```

The production site is generated in `dist/`; `pnpm preview` serves that build locally.
